```bash
# → API running at http://localhost:8000
# → Open dashboard.html in browser
```

## 🔄 How GRACE Integration Works

`grace_processor_real.py` is fully integrated into `api.py`.

When you run `/analyze`:
1. It checks if `data/grace/*.nc` exists
2. **If YES** → uses real GRACE data (process_grace_real)
3. **If NO**  → uses simulation based on published literature values

You do NOT need to change any code. Just place the `.nc` file in `data/grace/`
and restart the server — it automatically switches to real data.
