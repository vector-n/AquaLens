From a Map Click to a Complete Drilling Decision

01
📍
اختر الموقع
Select Location
اضغط على الخريطة التفاعلية أو أدخل إحداثيات GPS مباشرة. النظام يستقبل أي موقع داخل نطاق الدراسة فورياً.
02
🛰️
تحليل الفضاء
Satellite Analysis
يستدعي النظام بيانات GRACE-FO وصور Sentinel-2 ونموذج الارتفاع SRTM لاستخراج ٢٥+ ميزة جيولوجية للموقع تلقائياً.
03
🤖
الذكاء الاصطناعي
AI Prediction
ثلاثة نماذج XGBoost مترابطة تُصنّف الطبقة المائية، تتنبأ بالعمق المُعدَّل لـ٢٠٢٦، وتُقدّر الإنتاجية — مع قيود جيولوجية صارمة.
04
⚙️
تقييم التكوينات
Formation Evaluation
تطبيق معادلة أرشي على سجلات الآبار القريبة لاستخراج المسامية والتشبع المائي — منهجية مأخوذة من هندسة البترول للمرة الأولى في المنطقة.
05
📊
لوحة النتائج
Results Dashboard
واجهة واضحة تعرض الطبقة المستهدفة، العمق بنطاق عدم اليقين، الإنتاجية المتوقعة، مستوى الخطورة، ومواصفات الحفر الكاملة.
06
📄
تقرير PDF رسمي
Official PDF Report
بنقرة واحدة، يُنتج النظام تقريراً ثنائي اللغة (عربي/إنجليزي) جاهزاً للتقديم لسلطة المياه أو شركة الحفر.
